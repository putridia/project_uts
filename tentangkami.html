Inforkampus

(Informasi tentang Kampus)

Website Inforkampus ini adalah website penyedia informasi tentang kampus yang ada di pulau jawa dengan metode memberikan kemudahan kepada pengguna untuk mencari website resmi dari suatu kampus sehingga dapat memudahkan semua pihak untuk mencari informasi terkait kampus tertentu dan keunggulan website ini adalah dapat menyalurkan langsung ke website resmi kampus yang ada di pulau jawa.

InforKampus website is a website that provides information about campuses is on the island of Java with a method of making it easy for users to look for the official website of a campus so that it can make it easier for all parties to find information related to a particular campus and the advantages of this website is to be able to channel directly to the official campus website on the island of Java.
